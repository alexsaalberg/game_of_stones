# Game Of Stones

## Controls
* Use WASD to move. 
* Space to bounce
* Use arrow keys to change camera
* Hold right mouse to change camera
* Hold left, drag, then release to make selection
	* V to delete all in selection
	* B to fill selection
	* C to clear selection

## Naming Conventions
* Variables (local and member)
  * snake_case 
* Classes
  * PascalCase
* Functions
  * camelCase
